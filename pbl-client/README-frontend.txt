PBL0101 Frontend (Simple Vue via CDN)
- index.html : open this file in a browser (no build needed)

How to use:
1. Ensure backend is running on http://localhost:3000
2. Open pbl-client/index.html in your browser (double-click or open from VS Code with Live Server)
3. You can register, login, post, view profile, and see admin dashboard (simple demo)
