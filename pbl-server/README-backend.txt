PBL0101 Backend (Express + SQLite)
Files:
- server.js : main server
- package.json : npm config
- pbl.db : created automatically after first run

How to run (Windows, in VS Code terminal):
1. Open terminal in this folder (pbl-server)
2. Run: npm install
3. Run: node server.js
4. Server will listen on http://localhost:3000
